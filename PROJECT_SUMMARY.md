## 🔧 Customization Guide

### Change Colors:
Edit `tailwind.config.js` → colors section

### Update Content:
Edit respective section files in `/sections/` folder

### Add Pages:
Create new folder in `/app/` with `page.tsx`

### Modify Layout:
Edit `app/layout.tsx` for global changes

### Update Styles:
Edit `styles/globals.css` for global styles
