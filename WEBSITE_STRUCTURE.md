## 📱 MOBILE VIEW

All pages adapt to mobile with:
- Hamburger menu (☰)
- Stacked layouts (no side-by-side)
- Touch-friendly buttons
- Optimized images
- Easy scrolling

```
Primary Blue (#0c5a8a):
├── Headers
├── Navigation
├── Important text
└── Primary buttons

Aqua Green (#13c2c2):
├── Accents
├── Highlights

├── Backgrounds

└── Text
```

## 🎯 Call-to-Action Locations

1. **Hero Section** - Explore Services, Contact Us
2. **Services Cards** - Learn More links

4. **Contact CTA** - Get In Touch, WhatsApp, Call
5. **Header** - Call Now button
6. **Footer** - Contact links

## 📏 Section Spacing

- **Section Padding:** 64px top/bottom (desktop), 40px (mobile)
- **Container Width:** Max 1280px, centered
- **Grid Gaps:** 24px between cards
- **Card Padding:** 24px internal padding
